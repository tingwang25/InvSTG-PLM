# InvSTG-PLM for TrafficStream dataset.

### Data
You need to unzip the TrafficStream zip data from the provided [link](https://drive.google.com/file/d/1P5wowSaNSWBNCK3mQwESp-G2zsutXc5S/view?usp=sharing) and put all the files: `20XX.npz` in folder `district3F11T17/finaldata` into `InvSTG-PLM_TrafficStream/data`.

### Train
```bash
cd InvSTG-PLM/InvSTG-PLM_TrafficStream/src 
bash '../scripts/trafficstream.sh'
```
